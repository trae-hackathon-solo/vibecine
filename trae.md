# TRAE DEMO WALL Submission (English)

## Project

- Name: VibeCine
- Track: PixVerse Video Track (Hackathon MVP)
- One-liner: A shot-based video creator that turns a story prompt + character image into a 30s+ story via PixVerse, inside a visual flow workspace.

## Creation Story (Why did you build this? What problem does it solve?)

Most text-to-video demos break down when you try to generate a longer narrative: scenes feel disconnected, the main character’s identity drifts between shots, and a single long prompt is hard to control or regenerate safely.  

We built VibeCine to make video generation “shot-first”. Users start with a character reference image and a story prompt, then TRAE helps generate an editable storyboard (3-5 shots for the demo). Each shot can be refined and regenerated independently with PixVerse, while we inject lightweight context from the previous shot into the next prompt to improve continuity.  

This workflow makes the process reliable for a hackathon demo: we can show real iteration (edit one shot, regenerate one shot) and still guarantee a final story duration of at least 30 seconds by tuning per-shot duration.

## Key Highlights (1-5 items, max 30 chars each)
